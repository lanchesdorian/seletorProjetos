O HTML5 é a versão mais recente do HTML e suporta novas funcionalidades de linguagem de marcação, como multimídia, novas tags, elementos e novas APIs. O HTML5 também suporta áudio e vídeo. HTML não oferece suporte nativo a áudio e vídeo. HTML oferece suporte nativo a áudio e vídeo.

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais. O CSS trabalha em conjunto com o HTML, que define a estrutura do conteúdo, enquanto o CSS adiciona estilo e apresentação.

O que é a propriedade Display em CSS



Uma propriedade CSS é um atributo que define o estilo ou comportamento de um elemento HTML. Elas permitem controlar diversos aspectos visuais, como cor, tamanho da fonte, espaçamento, bordas, etc. Cada propriedade possui um nome e um valor, que determinam como o elemento será exibido no navegador.

Display Características
Block faz com que um elemento HTML seja renderizado como um bloco, ocupando toda a largura disponível do elemento pai e iniciando em uma nova linha.
Inline é exibido na mesma linha que outros elementos, ocupando apenas o espaço necessário para seu conteúdo (o que é diferente de block, que vai ocupar todo o espaço disponível, mesmo que sobre espaço após o seu conteúdo).
Inline-Block Permite que elementos fiquem lado a lado (como inline) e também aceitem definições de largura (width), altura (height), margem e preenchimento (padding) (como block).
Flex cria um "flex container" que permite um alinhamento flexível e distribuição de seus elementos filhos (flex items). Ele oferece controle sobre o posicionamento e dimensionamento dos itens ao longo de um eixo principal e um eixo transversal.
